A — Modal centrado con overlay

El dialog flota sobre la página con fondo oscurecido. Foco total en el formulario. Es el patrón de shadcn/ui Dialog — familiar, ampliamente entendido. Para formularios complejos, el modal puede ser más ancho o tener secciones internas con scroll.

B — Sheet lateral (drawer desde la derecha)

El formulario desliza desde la derecha sin tapar completamente el contenido. La tabla detrás sigue visible (aunque inactiva). Más espacio vertical para formularios largos como remitos. Es el patrón de shadcn/ui Sheet.

C — Modal para simples, Sheet para complejos

Formularios con pocos campos (asignar, registrar renovación, nuevo artículo) usan modal centrado. Formularios con secciones múltiples (remito con artículos y peajes, nuevo pedido con picker de artículos) usan sheet lateral. Cada patrón donde rinde más.
