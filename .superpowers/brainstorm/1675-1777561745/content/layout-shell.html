Especificaciones del shell

Sidebar
· Ancho fijo: 220px
· Fondo: white (distinto al fondo de página slate-50)
· Grupo activo expandido, el resto colapsado
· Item activo: fondo blue-50 + borde derecho blue-600 (2px)
· Íconos emoji o Lucide React a elección
· Footer: avatar inicial + nombre de usuario
Área de contenido
· Top bar: 44px, breadcrumb grupo › página
· Fondo: slate-50 (diferencia visual con cards)
· Page header: título 20px/700 + subtítulo opcional
· Botón de acción primaria: top-right del page header
· Cards y tablas sobre fondo blanco con borde slate-200
· Padding de contenido: 20px
